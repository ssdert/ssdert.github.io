TheFork

The Fork is one of the biggest application for booking restaurants and the 3th best around the world. The objective of this project was to redesign, prototype and test a new UI / UX concept that position the application as the best of all.

Restaurant page, it allows you to quickly browse the information of the restaurant by tapping on the top bar.

Home page, it provides you the most useful information all at once and it should help you to find a nice restaurant for tonight.

Location and filter, these features can help the user to pick the most suitable restaurant for him in the shortest time.

Profile page, you can manage all your personal information and check all your history bookings.

Notes.

This is our first UX project in SPD, through this project, we were able to practice more advanced prototyping techniques in XD as well as basic UX design and teamwork.
